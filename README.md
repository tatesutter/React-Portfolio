## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/react-portfolio.git
   ```

2. Navigate to the project directory:
   ```sh
   cd react-portfolio
   ```

3. Install dependencies:
   ```sh
   bun install
   ```

4. Start the development server:
   ```sh
   bun run start
   ```
